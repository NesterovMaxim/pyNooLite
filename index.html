Packages for working with noolite hardware

NooLite is a hardware for wireless light control. This software works with modern NooLite USB sticks (PC118, PC1116, PC1132) and easily can be used for making a smart home.

Description

This software was developed for working with NooLite USB sticks under GNU/Linux. Supported next sticks: PC118, PC1116, PC1132 and probably more. The software consis of a python class and example that used as a executable /usr/bin/noolite in some of packages.

Download

Source code
Central github source hosting for noolite software. You can clone reposotory by git clone https://github.com/Sicness/pyNooLite.git or just download a last archive

Ubuntu PPA
NooLite PPA repositories for Ubuntu Linux distrs. For install from PPA do:
sudo add-apt-repository ppa:ctolbhuk/noolite
sudo apt-get update
sudo apt-get install pynoolite

PyPI package
NooLite python package. PyPI is a Python Package Index is a repository of software for the Python programming language. Here is a python class for working with NooLite. If you have installed PyPI then for install noolite package just do: pip install noolite.
